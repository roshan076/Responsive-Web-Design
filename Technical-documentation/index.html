Introduction

Python is a high-level, interpreted, and general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Python provides a comprehensive standard library that includes modules and functions for various tasks, such as file I/O, system calls, and data manipulation. It is known for its simplicity and ease of learning, making it popular choice for both beginners and experienced programmers.

What you should already know

This guide assumes you have the following basic background:

  • A general understanding of programming concepts.
  • Familiarity with basic computer operations.
  • Experience with another programming language can be helpful but is not required.
Python and Other Languages

Python and languages like Java or C++ differ in several ways:

  • Typing: Python is dynamically typed, meaning you do not need to declare the data type of a variable explicitly. In contrast, language like Java use static typing.
  • Syntax: Python syntax is designed to be redable and straightforward. Unlike C++ and Java, Python does not use curly braces for blocks of code but relies on indentation.
  • Memory Mangement: Python handles memory management automatically trough garbage collection, unlike C++, which requires explicit memory management.
Hello World

To get started with writing Python, open a Python environment and write your first "Hello World" code:

print("Hello World")

Run the code in a Python interpreter to see the output.

Variables

Variables in Python are used to store data. The names of variables are called identifiers and follow specific rules:

  • An identifier must start with a letter (a-z, A-Z) or an underscore (_), and can be followed by letters, digits(0-9), or underscores.
  • Python is case-sensitive, so Variable, variable, and VARIABLE are considered different identifiers.
Declaring Variables

In Python, you can declare a variable by simply assigning it a value:

x = 42 y = "Hello"

Python uses dynamic typing, so you don't need to specify the data type.

Variable Scope

Variables can be declared in differnet scopes:

  • Global Scope: Variables declared at the top level of a script or module are global and accessible throughout the module.
  • Local Scope: Variables declared inside a function are local to that function.

Example:

def my_function():     local_var = "I am local"     print(local_var) gloabl_var = "I am global" print(global_var) my_function()
Constants

Python does not have a built-in constant type. However, you can create constatns by convention using uppercase names:

PI = 3.14

Constants are not enforced by the language, they are only a convention to indicate that a variable should not be changed.

Data Types

Python includes several built-in data types:

  • int: Integer values (e.g, 42)
  • float: Floating-point values (e.g., 3.14159)
  • str: String vlaues (e.g., "Hello")
  • bool: Boolean values (True or False)
  • list: Ordered, mutable collection (e.g., [1, 2, 3])
  • tuple: Ordered, immutable collection (e.g., (1, 2, 3))
  • dict: Unordered collection of key-value pairs (e.g., {"key": "value"})
  • set: Unordered collection of unique elements (e.g., {1, 2, 3})
if...else Statement

Use the if statement to execute code based on a condition. The else clause executes if the condition id false. Example:

if condition:     statement_1 else:     statement_2

You can also use elif to test multiple conditions:

if condition_1:     statement_1 elif condition_2:     statement_2 else:     statement_last
While Statement

A while loop repeatedly executes a block of code as long as a specified condition is true:

while condition:     statement

Example:

n = 0 x = 0 while n < 3:     n += 1     x += n
Function Declarations

A function in Python using the def keyboard followed by the function name, parameters in parentheses, and a block of code:

def function_name(parameters):     # function body     return value

Example:

def square(number):     return number * number
